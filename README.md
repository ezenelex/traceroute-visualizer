Running

1. Download the code and extract into a folder.
2. In a command prompt, start with this command in the folder:
> npm run devStart
3. Open the app at localhost:5000 in the browser

Change which site traceroute visualizes

1. Open server.js
2. Change the URL in line 12 to anything you'd like to visualize
